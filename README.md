# .github Community Health Files

This repository contains the shared default community health files for the **Notebook.Me** organization.

GitHub automatically uses supported files from this repository across public repositories in the organization when those repositories do not define their own versions.

---

## 📁 What This Repo Is For

Use this repository to manage organization-wide defaults such as:

* `CODE_OF_CONDUCT.md`
* `CONTRIBUTING.md`
* `SECURITY.md`
* Issue templates
* Pull request templates
<!-- * Funding configuration -->
* Shared documentation standards

---

## 🚀 Why It Exists

Maintaining these files in one place helps:

* Keep all repositories consistent
* Save setup time for new projects
* Improve contributor experience
* Standardize reporting and contribution flows

---

## 🧩 How GitHub Uses This Repo

If a repository inside the organization does **not** contain its own supported community files, GitHub can fall back to the defaults stored here.

Repository-specific files will always override shared defaults.
